# Arch Linux + Hyprland Dev Setup

Configuração completa de um ambiente de desenvolvimento moderno utilizando **Arch Linux** e **Hyprland**, feita do zero e documentada passo a passo.

Este repositório acompanha toda a evolução do sistema: desde a instalação base até um setup totalmente funcional e produtivo.

---

## 📚 Sobre este projeto

Este projeto faz parte do meu processo de aprendizado.

Sou iniciante no ecossistema **Arch Linux** e no uso do **Hyprland**, e estou documentando toda a jornada — incluindo erros, ajustes e decisões ao longo do caminho.

Durante o desenvolvimento, utilizo ferramentas de IA generativa como apoio para pesquisa, explicações e sugestões. No entanto, busco sempre entender, testar e validar as soluções antes de aplicá-las.

O objetivo não é apenas chegar a um setup final, mas construir conhecimento sólido e um material útil para referência futura (minha e de outras pessoas em situação semelhante).

---

## 🎯 Objetivo

* Construir um ambiente Linux totalmente personalizado
* Aprender profundamente o ecossistema do Arch Linux
* Explorar o compositor Wayland **Hyprland**
* Criar um setup reproduzível
* Servir como documentação pessoal e referência pública

## 🧠 Filosofia do Projeto

Este projeto segue alguns princípios:

* **Aprendizado ativo:** tudo é feito manualmente antes de automatizar
* **Documentação clara:** cada passo deve ser compreensível
* **Reprodutibilidade:** o setup deve ser replicável
* **Evolução contínua:** o sistema será refinado ao longo do tempo
